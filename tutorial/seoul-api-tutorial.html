예시: 7559776558746f74...

STEP 03

API 주소(URL) 구조 이해하기

API를 호출하는 주소는 마치 편지 주소처럼 구조가 있습니다.

http://openapi.seoul.go.kr:8088/{API키}/json/neisSchoolInfoJS/{시작번호}/{끝번호}/{학교명}

🔑 API키

발급받은 인증키를 넣습니다.
이걸로 "나 허가받았어요" 확인

📋 시작/끝번호

데이터를 몇 번째부터 몇 번째까지 받을지 지정합니다.
전체: 1 ~ 610

🔍 학교명

특정 학교를 검색할 때 씁니다.
전체 조회는 공백(%20)을 넣습니다.

실제 예시: http://openapi.seoul.go.kr:8088/나의키/json/neisSchoolInfoJS/1/10/%20
→ 전체 학교 중 1~10번째 데이터를 JSON 형식으로 받아옵니다.

STEP 04

받아온 데이터 생김새

API가 돌려주는 데이터는 이렇게 생겼습니다. JSON 형식이라고 부릅니다.

// API 응답 예시 (JSON)
{
  "neisSchoolInfoJS": {
    "list_total_count": 610,
    "RESULT": {
      "CODE": "INFO-000",
      "MESSAGE": "정상"
    },
    "row": [
      {
        "SCHL_NM": "광진초등학교",
        "SCHL_KND_NM": "초등학교",
        "FNDN_SE": "공립",
        "ROAD_NM_ADDR": "서울시..."
      }
    ]
  }
}

주요 필드 설명

필드명의미
SCHL_NM학교명
SCHL_KND_NM학교종류
FNDN_SE공립/사립
CMPTNC_OGNZ_NM교육지원청
ROAD_NM_ADDR도로명주소
FNDN_YMD설립일(YYYYMMDD)
HMPG_ADDR홈페이지 URL
✅ RESULT.CODE가 "INFO-000"이면 정상입니다. 다른 코드면 에러입니다.

⚠️ 중요 보안

API 키는 어디에 두어야 할까요?

API 키가 노출되면 남이 내 명의로 데이터를 마음대로 가져갑니다. 반드시 숨겨야 합니다.

❌

절대 하면 안 되는 것

  • 브라우저 코드(프론트엔드)에 직접 넣기
  • GitHub에 .env 파일 올리기
  • 카카오톡, 이메일로 공유하기
  • URL에 그냥 넣어서 공개하기
✅

올바른 방법

  • .env.local 파일에 저장 (Git 제외)
  • 서버(API Route)에서만 읽기
  • Vercel 환경변수 설정창에 입력
  • 클라이언트에 절대 내보내지 않기
# .env.local 파일 (이 파일은 Git에 올라가지 않습니다)
SEOUL_API_KEY=여기에_발급받은_키_입력

서버 프록시 패턴이란?

클라이언트가 직접 외부 API를 호출하면 키가 노출됩니다.
서버를 중간에 두어 키를 숨기는 방법입니다.

👤
사용자 브라우저
API 키 없음
→
⚙️
Next.js 서버
app/api/seoul/route.ts
🔑 API 키 보관
→
🏛️
서울 공공 API
openapi.seoul.go.kr:8088
// app/api/seoul/route.ts — 서버에서만 실행
const apiKey = process.env.SEOUL_API_KEY  // 환경변수에서 읽기
const url = `http://openapi.seoul.go.kr:8088/${apiKey}/json/...`
const res = await fetch(url)                   // 서버가 호출

Vercel로 인터넷에 공개하기

내 컴퓨터에서만 보이던 서비스를 전 세계 누구나 볼 수 있게 만드는 과정입니다.

1

vercel.com 접속 후 GitHub 로그인

GitHub 계정으로 로그인하면 저장소가 자동 연결됩니다.

2

저장소 선택 후 배포

philosophyAIEDU/260606seoul 선택 → Deploy 클릭

3

환경변수 설정

Settings → Environment Variables → SEOUL_API_KEY 추가

🚀

자동 배포

GitHub에 코드를 올리면 Vercel이 자동으로 새 버전을 배포합니다. 별도 작업 불필요!
